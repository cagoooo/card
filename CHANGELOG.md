# 變更紀錄 (Changelog)

所有關於「卡牌工坊」(Card Workshop) 專案的顯著變更都將記錄於此檔案中。

## [1.0.0] - 2026-04-12

### 🚀 新增功能 (Features)
- **資料持久化 (Phase 1)**
  - 全新實作 `useLocalStorage`，自動儲存設計數據，關閉分頁不遺失。
  - 支援將專案匯出為 JSON，以及將先前的設計匯入恢復。
- **使用者體驗提升 (Phase 2)**
  - 大富翁圖板支援 縮放控制 (`Ctrl + 滾輪` 或 UI 介面)，並導入 `framer-motion` 流暢動畫。
  - 圖板支援格數切換：20格、32格、**40格**、52格。
  - 新增 5 大**卡牌預設模板**（標準撲克、奇幻冒險、中文主題、教學數字卡、色彩學習）。
  - 花色列表現已支援 **@dnd-kit 拖放排序**，自由調整順序。
  - 支援動態切換列印尺寸（撲克牌、橋牌、塔羅牌）。
- **進階設計功能 (Phase 3)**
  - **本地專案管理 (IndexedDB)**：全新的 `/projects` 專案大廳，可在單一設備管理並儲存多套專案。
  - **圖像上傳**：卡牌現支援自動壓縮 (≤150KB) 的本地圖片插畫上傳。
  - **卡牌風格**：加入 5 種卡面視覺主題切換（包含經典白、奇幻羊皮、科幻霓虹、水墨風、木板質感）。
  - **Emoji 快選圖標**：現在可以為大富翁的每個專屬格子加上醒目的圖標。
  - **卡背設計室**：全新 `/cards/back` 路由，支援自訂邊框、底色組合與 4 種精美底紋樣式（點陣、格子、斜線、交叉），一次列印一整頁。
- **教育與列印應用 (Phase 4)**
  - **精準列印排版 (Print Layout)**：新增完整 A4 列印排版系統 (`/print-layout` 路由)。
  - **動態尺寸計算**：支援撲克牌(3x3)、橋牌(3x3)、塔羅牌(2x2) 自動分頁與 CSS grid 網格排列。
  - **裁切輔助**：實作實體紙牌列印專用裁切十字輔助線 (`crop marks`)，利於教師手工裁切製作。

### 🐞 修正 (Fixes)
- 修正 GitHub Pages 部署時的基礎路徑 (Base Path) 問題，現已強制轉換為相對路徑 `./` 以完美相容。

### ⚙️ 環境與架構 (Chores)
- 正式加入 `@dnd-kit`、`browser-image-compression`、`idb-keyval`、`framer-motion` 依賴。
- 專案版號首度定立為 `1.0.0`。
